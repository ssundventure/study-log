
### 💡 CSS에서 상대적, 절대적 개념

| 개념   | 뜻                         | CSS 예시                                    |
| ------ | -------------------------- | ------------------------------------------- |
| 상대적 | **기준**이바뀌면 값도 바뀜 | relative : 원래 위치에서 이동               |
| 절대적 | **기준**이 고정됨          | absolute : 부모 기준으로 정확한 위치에 고정 |
