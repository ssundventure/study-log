# useEffect

## 정의 - useEffect는 무엇인가 ?

- useEffect는 렌더링 이후 원하는 시점에 코드가 실행되게 해주는 함수

## 이유 - 왜 사용하는가 ?

- 렌더링마다 반복 실행되는 것을 막고, 원하는 시점에만 실행하기 위해 사용한다.
- 렌더링 후에 실행되어야하는 코드를 분리하기 위해 사용한다.
  

## 조건 - 언제 사용하는가 ?

- 데이터 가져오기(API 등) 같은 작업이 필요할 때 사용한다.
- 렌더링 이후에 실행되어야 하는 코드일 때 사용한다.
- 특정 값이 변경될 떄만 실행하고 싶을 때 사용한다.

## 구조 - 어떻게 동작하는가?

1. 컴포넌트가 렌더링된다.
2. 렌더링이 끝난 후 useEffect가 실행된다.
3. dependency 배열에 따라 실행 시점이 결정된다.

## 실전 - 내 코드에서는 어디에 쓰였는가 ?

- localforage에서 posts 데이터를 가져올 때 사용된다.
- 처음 렌더링 시 데이터를 불러와 state를 업데이트하는데 사용된다.


## 핵심 
- useEffect는 렌더링 이후에 실행되는 코드이다.
