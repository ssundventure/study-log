# props

## 정의 - props는 무엇인가 ?

- props는 부모 컴포넌트가 자식 컴포넌트에게 데이터를 전달할 때 사용하는 것이다.

## 이유 - 왜 사용하는가 ?

- 부모의 데이터를 자식에게 전달해서 같은 데이터를 기준으로 UI를 보여주기 위해 사용한다.

## 조건 - 언제 사용하는가 ?

- 부모의 값이 자식 컴포넌트에도 필요할 때 사용한다.
- 자식이 부모의 데이터를 받아 UI를 그려야 할 떄 사용한다.
- 자식에서 부모 state를 변경해야한다면 부모의 함수를 props로 전달한다.

## 구조 - 어떻게 동작하는가?

- 부모 컴포넌트가 props를 통해 값을 전달하면 자식은 그 값을 받아 UI를 그린다.

## 실전 - 내 코드에서는 어디에 쓰였는가 ?

- FilmColumn 컴포넌트는 부모로부터 posts와 align 을 props로 받는다.
- props는 구조분해로 받아 사용한다.
- 전달 받은 posts 와 align은 포스트를 출력하거나 스타일을 다르게 적용하는데 사용된다.

## lifting state up

- 여러 컴포넌트가 같은 state를 사용해야 할 때 그 state를 공통 부모로 올려서 관리하는 것

## 특징

- props는 읽기 전용이며, 자식 컴포넌트에서 직접 변경할 수 없다.

## 핵심

- props는 부모 -> 자식으로 데이터를 전달하는 도구이다.
